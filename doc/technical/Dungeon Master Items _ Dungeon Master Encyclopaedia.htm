Dungeon Master Items

Mon, 2005-10-24 02:00 — Christophe Fontanel

This section has an exhaustive list of items available in the game. Some of the items cannot be found in the dungeon, they are only accessible by using editing tools like DMute for PC v1.4.
The locations of items are based on the Dungeon Master for PC dungeon. There may be some differences in other versions of the game.
The characteristics of items were extracted from Dungeon Master for Atari ST by using the DMExtract v1.01 and CSBEdit v1.0 for PC tool by rain` (rain@www.qwplayers.org).
Notes:
Action Difficulty: The amount of experience needed in the specified action's skill to be able to use the action (see the Technical Documentation - Dungeon Master and Chaos Strikes Back Skills and Statistics page).
Armor Strength: The armor value (0..200) associated with the item.
Sharp Resistance: A 0..7 value which is used when monsters have a "sharp" attack. Sharp attacks use the following format to calculate armor: (Sharp Resistance / 7) * Armor Strength, so if the Sharp Resistance value is 0, a creature having a Sharp attack will ignore your armor.
Damage: The base number used to calculate how much damage your attack does.
Distance: This value determines how far the item will go when thrown. If a weapon is used to "Shoot", this will be a part of how far the item being shot will travel. The farther the projectile goes, the more damage it does (Damage is decreased as it flies).
Shoot Damage: The amount of damage associated with fired projectiles.
Class / Delta energy: The weapon class and delta energy (only for bows and slings) lost for each dungeon tile the missile travels.
